#项目介绍

DrawingBoard是一个画笔涂鸦功能，几行代码实现，支持自定义

如果觉得对你有帮助，不妨点个★star支持鼓励我~

** 本库将持续维护下去~~~~**

使用说明

在项目module下的gradle中添加以下依赖：
仓库：
```
allprojects {
		repositories {
			maven { url 'https://jitpack.io' }
		}
	}
```
compile 'com.github.w513209188:DrawingBoard:v1.0.0'
